Blog 再起

2018-05-20
  • Diary

之前

之前也有過幾次「覺得應該要來寫Blog」的想法。也不是不曾動手過，試過幾次，換過幾個不同平台

最終還是都被 惰性大魔王 擊敗。

曾經

例如曾經用 Google 的 Blogger 寫了幾篇流水帳
不太熟悉地東調西改網頁模板，也去找了 SyntaxHighlighter 加入網站來顯示程式碼
以為自己會多有毅力寫下去這樣 XD

荒廢了很久一段時間，後來又看到人家分享，使用靜態網站產生器 (Static Site Generators)
支援 Markdown 語法產生 Html，再上傳到 Github，免費又方便！
(謎：Blogger 也是免費的啊)

試過

之前試過兩款：Jekyll、Hugo

  • Jekyll
    • 是用 Ruby 寫的，記得之前在本機上安裝環境上遇到一些問題還滿不知所措的
  • Hugo
    • 剛剛查是 Go 寫的，細節已經完全沒有印象
      (剛看本機上相關檔案都已經是 2年前 的事，剛好是前一份工作離職時 以為自己有心嘗試些新玩意)

總之呢呃每次嘗試之間，都間隔了段不短的時間 ……

最近

這次則是最近在 FB 上看到 Alan Tsai 大大分享的 Wyam系列文章
覺得可以再來試看看
所以就嘗試用了 Hexo 搭配上傳 Github 來寫寫文章。(咦什麼神展開)

Wyam 是用 .Net 開發的 SSG專案，理論上我寫 C# 應該要感到比較熟悉才對
不知為何陰錯陽差，後來看到 Hexo 相關介紹文章反而被吸引了過去 XD

查資料過程中發現，這種 Static Site Generator 還真不少
甚至有個 StaticGen 網站整理出 Github 上各式 SSG 專案，還可以依照 開發語言 或 Github星星、Fork數 等條件過濾。

Hexo

其中 Hexo 是無意中看到的，聽說 開發者是台灣人，更增添了一些親切感

安裝過程就不多提了，網路上很多教學文章

過程中也是遇到大大小小狀況，邊 Google 邊排除
因為還不習慣 blog 撰寫，也沒有截圖或作筆記什麼的

就大概列一下摸到的東西 (有些甚至跟這次主題無關 XD)

  • Cmder
  • Cmder Git 設定
  • cmder-powerline-prompt
  • AnonymousPro font
  • Powershell - Install-Module posh-git
  • VSCode - 終端機改用 Cmder
  • VSCode - 雜七雜八設定調整
  • Hexo
  • Hexo Theme - Chan

前幾項來亂的其實是看到人家教學文章提到一些環境調教，
順便照著做，然後踩坑延伸出的問題排除步驟 …

最後結果是使用 VSCode 來作為主要撰寫工具

Markdown 語法支援
終端機整合
以及各式各樣的擴充功能

VSCode真是好棒棒！

總之

總之這次又生出了個 Hexo 版的 Hello Word Blog 文章

目前版面看起來沒有很理想，但是對前端很不熟，只好慢慢嘗試調整囉


希望這次會撐久一點
