Hexo 基本指令

2018-05-21

完整可以看這邊 Hexo Doc

這邊記一下 Hexo 常用的 terminal 指令

Init - 初始化目錄

沒有寫 folder 代表針對當前目錄

New - 新增文章

預設 /scaffolds/ 下有 3種 layout 讓你選

  • draft.md
  • page.md
  • post.md

Clean - 清除 publish 產生的檔案

Generate - 建置

Server - 啟動伺服器
在本機執行一個 web server，可以在 localhost:4000(預設) 預覽結果
文件有修改也會即時更新，在瀏覽器重新整理就可以看到變化

Deploy
依照 /_config.yml 裡面的設定部署至指定目的

大概4這樣
